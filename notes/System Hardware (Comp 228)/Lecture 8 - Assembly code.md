▶ **Direct addressing** when the address is provided in the instruction 
▶ **Immediate addressing** when the value being worked on is part of the instruction 
▶ **Indexed addressing** when the address to the start of an array is part of the instruction but the index is in a register 
▶ **Indirect addressing** when the address is of the value is in memory and the address of that memory location is part of the instruction 
	▶ You have the location of the treasure map (address included in the instruction), you need to get the treasure map (address of the data we want) before you can find the treasure (the data we want) 
▶ **Implied addressing** when main memory access is not involved
